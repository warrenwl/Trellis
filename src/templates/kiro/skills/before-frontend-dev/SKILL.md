# 前端开发准备

在开始开发任务之前，请先阅读前端开发指南。

请执行以下步骤：
1. 阅读 `.trellis/spec/frontend/index.md` 以了解可用的指南
2. 根据您的任务，阅读相关的指南文件：
   - 组件工作 → `.trellis/spec/frontend/component-guidelines.md`
   - Hook 工作 → `.trellis/spec/frontend/hook-guidelines.md`
   - 状态管理 → `.trellis/spec/frontend/state-management.md`
   - 类型问题 → `.trellis/spec/frontend/type-safety.md`
3. 了解您需要遵循的编码标准和模式
4. 然后继续执行您的开发计划

在编写任何前端代码之前，这一步是**必需的**。

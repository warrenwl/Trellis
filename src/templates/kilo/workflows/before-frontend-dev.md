阅读前端开发指南后再开始你的开发任务。

执行以下步骤：
1. 阅读 `.trellis/spec/frontend/index.md` 了解可用的指南
2. 根据你的任务，阅读相关的指南文件：
   - 组件开发 → `.trellis/spec/frontend/component-guidelines.md`
   - Hook 开发 → `.trellis/spec/frontend/hook-guidelines.md`
   - 状态管理 → `.trellis/spec/frontend/state-management.md`
   - 类型问题 → `.trellis/spec/frontend/type-safety.md`
3. 了解你需要遵循的编码标准和模式
4. 然后按照你的开发计划进行

这是在编写任何前端代码之前**必须**执行的步骤。

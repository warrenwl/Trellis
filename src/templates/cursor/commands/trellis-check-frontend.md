# 前端代码检查

检查您刚刚编写的代码是否符合前端开发指南。

执行以下步骤：
1. 运行 `git status` 查看修改的文件
2. 阅读 `.trellis/spec/frontend/index.md` 了解适用的指南
3. 根据您修改的内容，阅读相关的指南文件：
   - 组件修改 → `.trellis/spec/frontend/component-guidelines.md`
   - Hook 修改 → `.trellis/spec/frontend/hook-guidelines.md`
   - 状态修改 → `.trellis/spec/frontend/state-management.md`
   - 类型修改 → `.trellis/spec/frontend/type-safety.md`
   - 任何修改 → `.trellis/spec/frontend/quality-guidelines.md`
4. 根据指南检查您的代码
5. 报告任何违规项并进行修复
